# GasWaterDeliverySystem

A console-based Java application designed to manage and track gas and water deliveries.  
The system allows the registration of deliveries, sales tracking, and detailed listing of each delivery.  
Developed to practice Java fundamentals, object-oriented programming, and array management.

---

## Features

- Register new delivery departures  
- Record delivery returns (sold gas units)  
- Display all registered deliveries  
- Prevent invalid operations and ensure data consistency  
- Simple text-based interface  

---

## Technologies Used

- **Java**  
- **Eclipse IDE**  
- `java.util.Scanner`  
- `java.util.Locale`  
- Arrays (`Delivery[]`)
